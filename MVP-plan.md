## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Look |
| Shift | Run |
| Space | Jump |
| V | Toggle 1st/3rd person |
| E | Interact |
| C | Clear whiteboard |
| Escape | Menu |
